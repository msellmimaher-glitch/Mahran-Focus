# Maher Focus — version allégée

Cette version réduit la surcharge visuelle de l'écran Aujourd'hui.

## Changements principaux

- Écran Aujourd'hui limité à 4 tâches visibles en mode normal.
- Mode calme / journée difficile limité à 3 tâches.
- Les tâches restantes sont cachées derrière un simple résumé.
- Cartes plus sobres, moins de boutons, moins de texte.
- Barre de navigation plus courte : Jour, Capture, +, Objectifs, Plus.
- Couleurs plus douces et interface moins dense.

## Générer l'APK sans Android Studio

1. Envoyer tout le contenu du projet dans GitHub.
2. Aller dans Actions.
3. Lancer Build Maher Focus APK.
4. Télécharger l'artifact MaherFocus-debug-apk.
5. Décompresser le fichier téléchargé.
6. Installer app-debug.apk sur le téléphone Android.

## Note importante

Cette version est une V1 locale hors ligne. Elle n'a pas de cloud et pas de compte utilisateur.
