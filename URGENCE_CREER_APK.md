# Méthode la plus simple pour créer l'APK sans Android Studio

1. Dans GitHub, ouvre ton dépôt `MahrFocus`.
2. Clique sur **Code**.
3. Supprime l'ancien contenu si possible, ou crée un nouveau dépôt propre.
4. Téléverse tout le contenu de ce dossier, pas le dossier parent.
5. Va dans **Actions**.
6. Clique sur **Build Maher Focus APK**.
7. Clique sur **Run workflow**.
8. Quand c'est vert, télécharge **MaherFocus-debug-apk**.
9. Décompresse l'artifact et installe `app-debug.apk` sur ton téléphone.

Si GitHub demande l'autorisation d'installer une application inconnue sur Android, accepte seulement pour ton navigateur ou ton gestionnaire de fichiers.
